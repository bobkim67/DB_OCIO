## 시장영향 vs 원인 경계 규칙

```
1. 제목에 자산/시장 가격이 직접 명시 → 해당 자산 토픽
   예: "Oil prices climb" → 에너지_원자재
   예: "rupee 98/USD" → 환율_FX
   예: "금값 5311달러" → 귀금속_금

2. 제목에 가격 없이 원인만 → 원인 토픽
   예: "Iran war rattles Gulf" → 지정학
   예: "ECB 금리 동결" → 통화정책

3. 가격 + 원인 모두 → 가격(시장영향) 우선
   예: "Oil prices climb after Iran warns" → 에너지_원자재
```

## 통화정책 vs 달러_글로벌유동성 경계 규칙

```
- 중앙은행 주체 (Fed/ECB/BOJ/BOK/한은) + 정책결정/발언 → 통화정책
- 유로달러 시스템/글로벌 달러 부족/스왑라인 → 달러_글로벌유동성
- "달러 유동성 양호" + 한은 총재 → 통화정책 (주체가 중앙은행)
```
