## 2. 블로그 크롤러 (naver_blog.py)

### 기술 이슈 해결

1. **네이버 iframe 구조**: PC 메인은 mainFrame iframe, headless에서 불안정 → 모바일(m.blog.naver.com) 무한 스크롤로 전환
2. **ChromeDriver 타임아웃**: 장시간 세션 → localhost 연결 끊김 → 2단계 분리 (step1_collect_urls → step2_scrape_posts) + 100건 배치 단위 드라이버 재시작
3. **내부망 SSL**: `WDM_SSL_VERIFY=0` + `--ignore-certificate-errors`
4. **모바일 스크롤 한계**: 504건에서 더 이상 로드 안 됨 (전체 608건). 후속 incremental로 +6건 추가 → 510건

### 수집 통계

- 총 510건, 실패 0건
- 날짜 범위: 2024-01-03 ~ 2026-03-23
- 평균 본문: 2,504자, 최대 12,750자
- 카테고리: 200+ 종류

## 3. 블로거 경제관 분석

### 핵심 발견

- **학파**: 유로달러 학파 (Jeff Snider / Alhambra Investments 계열)
- **중심 변수**: 유로달러 유동성 → 엔화, 위안화, 원화, 금, 유가, 비트코인, 부동산 모두 종속
- **2025년 게임체인저**: 독일 부채 브레이크 폐기 + 유럽 은행 부활 = "유로달러 르네상스"
- **분석 방법론**: 내러티브 해체 + 가격 우선 + 미시구조 분석

### 팩트체크 패턴

- 수치 과장 10~50% 반복 (특히 전황 관련)
- 에스컬레이션 편향
- 구조 분석은 우수 (사모대출 POST가 가장 높은 신뢰도 ⭐⭐⭐)

- dataset 405: MOVE Index (6,493건, 2001~2026)
- dataset 399: LUATTRUU (6,337건, 2001~2026)
- dataset 419: EM Dollar Index / FXJPEMCS (4,098건, 2010~2026)
- dataset 420: UST 7-10Y TR / LT10TRUU (6,337건, 2001~2026)

### 레포 실패 데이터 확보 과정

FRED에서 시리즈 ID 변경/미존재 → NY Fed Primary Dealer Statistics API 탐색:
- `https://markets.newyorkfed.org/api/pd/list/timeseries.json` 에서 전체 시리즈 검색
- Treasury fails: `PDFTD-USTET` (인도), `PDFTR-USTET` (수취)
- seriesbreak `SBN2024` 필수
- 89건씩 수집 성공 (2024-07 ~ 2026-03)
