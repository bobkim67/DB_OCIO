## 2. 변경 상세

### 변경 1: 히트맵 기간선택 multiselect 제거

- **위치**: tabs[4] 하단, 주요 벤치마크 기간수익률 히트맵
- **Before**: `st.multiselect("기간 선택", ...)` 위젯으로 기간 동적 선택
- **After**: 고정 `['1M', '3M', 'YTD', '1Y', '3Y', '5Y']` 출력
- `st.columns` 레이아웃 제거, `st.markdown` 직접 출력으로 단순화

### 변경 2: Growth Rate 기준점 초기화

- **위치**: tabs[4] EPS & PE Ratio 추이 섹션, Growth Rate 모드
- **Before**: 롤링 growth rate `pe_s / pe_s.shift(n_shift) - 1` (각 시점마다 다른 기준)
- **After**: 선택 기간 시작점에서 0%부터 시작하는 indexed return
  ```python
  start_idx = max(0, n_md - n_shift - 1)
  pe_trimmed = pe_s.iloc[start_idx:]
  pe_indexed = (pe_trimmed / pe_trimmed.iloc[0] - 1) * 100
  ```
- 차트 x축도 `trimmed_dates = macro_dates[start_idx:]`로 선택 기간만 표시
- 모든 티커가 0%에서 출발 -> 상대 비교 용이

### 변경 3: 자산군별 벤치마크 수익률 다중 기간 칼럼

- **위치**: tabs[4] 자산군별 벤치마크 수익률 테이블 (42개 행)
- **Before**: 단일 `'기간변동'` 칼럼 (문자열 '-1.23%')
- **After**: 7개 기간별 수치 칼럼 `1D, 1W, 1M, 3M, 6M, 1Y, YTD`
- 칼럼 순서: `대분류, 중분류, 소분류, 지표, 현재값, 1D, 1W, 1M, 3M, 6M, 1Y, YTD, Source, dataseries, dataset_name, 최근기준일`
- **유형별 단위 포맷팅**:
  - 주식/채권지수/FX/원자재: `+2.31%`, `-1.23%`

- Source 배경색: Factset 파랑, Bloomberg 노랑, KIS 연두

## 4. 기술 노트

### Pandas Styler 제약

- `st.dataframe`의 Styler 지원이 제한적: `.bar()` 메서드는 문자열 칼럼에서 동작 불가
- 해결: 숫자 데이터를 문자열로 포맷한 후 `.map()`으로 텍스트 색상만 적용
- `.bar()` 인라인 바차트는 Streamlit 호환성 문제로 보류

### 행별 포맷터 설계

- `_row_type` Series로 각 행의 유형 태그 관리 (`return`, `bp`, `vol`, `econ`)
- `_make_env_formatter(row_types, src_data)` 함수로 유형별 포맷 문자열 생성
- 원화환산 토글 상태에 따라 `macro_env_data` vs `macro_env_data_active` 선택

### Forward-fill 패턴

- 대분류/중분류가 빈 문자열인 행은 이전 행의 값을 상속하는 계층 구조
- `_env_ff['_대분류'] = _env_ff['대분류'].replace('', np.nan).ffill()` 패턴으로 처리
- tabs[4], tabs[5] 모두 동일 패턴 사용
