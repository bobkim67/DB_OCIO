## 2. 변경 상세

### 2.1 DB 연동 Phase 1 (Tab 0, 1, 6)

**config/funds.py**
- `FUND_BM` dict 추가: 21개 펀드 → SCIP dataset_id/dataseries_id 매핑
- 현재 전 펀드 S&P 500 TR (dataset_id=24, dataseries_id=6) 기본값
- `ASSET_6CLASS` 리스트 추가

**modules/data_loader.py** — 신규 함수 7개, 수정 2개:

| 함수 | 용도 |
|------|------|
| `get_pandas_connection(db)` | pd.read_sql용 일반 커서 (DictCursor 호환 문제 해결) |
| `parse_data_blob(blob, currency)` | SCIP longblob JSON 파싱 (3가지 형태) |
| `load_scip_bm_prices(...)` | BM 지수 시계열 (SCIP back_datapoint) |
| `load_usdkrw_from_scip(...)` | USD/KRW 환율 (dataset_id=31) |
| `_classify_6class(row)` | AST_CLSF_CD_NM → 8분류 매핑 |
| `load_fund_holdings_classified(...)` | 보유종목 + 분류 + 콜론 그룹핑 |
| `_extract_fund_code_from_item_cd(...)` | 모펀드 ITEM_CD → 펀드코드 추출 |
| `load_fund_holdings_lookthrough(...)` | 모펀드 하위 종목 전개 (look-through) |
| `load_fund_nav_with_aum(...)` | NAV + AUM 시계열 (확장) |

기존 수정:
- `load_scip_prices()` — dataseries/start_date 필터 추가
- `load_all_fund_data()` — nav 데이터 포함

**prototype.py** — Tab 0/1/6 DB 교체:
- import + DB_CONNECTED 플래그 + 7개 @st.cache_data 래퍼
- Tab 0: NAV, BM, Holdings, AUM 모두 DB (fallback: mockup)
- Tab 1: Holdings, AP 비중 DB (MP는 Phase 2)
- Tab 6: Fund summary DB
- Tab 2: ValueError 수정 (nav_data 길이 불일치)

### 2.2 자산 분류 체계 변경: 6분류 → 8분류

```
Before: 국내주식, 해외주식, 국내채권, 해외채권, 대체투자, 유동성
After:  국내주식, 해외주식, 국내채권, 해외채권, 대체투자, FX, 모펀드, 유동성
```

- `_classify_6class()` 수정: 모펀드 → '모펀드' (기존 '유동성'), 달러선물/NDF → 'FX'
- `ASSET_CLASSES`, `ASSET_COLORS`, `ASSET_TO_MARKET_INDICATORS` 모두 8분류로 확장
- 모든 hardcoded 비중 리스트 8-element로 업데이트

### 2.3 모펀드 Look-through

- **문제**: 모펀드 ITEM_CD가 `032280007J48` 형태 (IMC_CD + 펀드코드)
- **해결**: `_extract_fund_code_from_item_cd()` → 뒤 5자리 추출 → 하위 펀드 조회
- 상단 바에 토글 배치 (모펀드 편입 펀드에서만 표시)
- 검증: 07J34(MySuper 성장형) 모펀드 2개 → 18개 하위 종목 정상 전개

### 2.4 펀드 선택기 개선

- **Before**: `{펀드명} ({AUM}억)` — AUM 내림차순
- **After**: `{코드}  {펀드명}` — 코드 오름차순
- 상단 바 레이아웃: 그룹 → 펀드 → Look-through → 펀드정보 → 로그아웃 (5컬럼)

### 2.5 자산군 정렬 적용

- `ASSET_CLASS_ORDER` dict 도입
- Tab 0 전체 보유종목: 자산군 순서 → 비중 내림차순
- Tab 1 자산군별: `ASSET_CLASS_ORDER` 순서
- Tab 1 종목별: 자산군 순서 → 비중 내림차순
- 순서: 국내주식 → 해외주식 → 국내채권 → 해외채권 → 대체투자 → FX → 모펀드 → 유동성
