## 2. R Shiny PA 로직 vs Python 현재 구현 비교

### 2.1 핵심 발견: modify_unav_chg 합산 = 기준가 변동 (완벽 일치)

```
08K88 20260305:
  modify_unav_chg 합계: 41.099777
  기준가 변동 (MOD_STPR - PDD_CHNG_STPR): 41.099776
```

수정기준가는 종목별 modify_unav_chg의 정확한 합산이므로, 종목별 분해가 가능.

### 2.2 pl_gb별 기준가 분해 (20260305 08K88)

| pl_gb | modify_unav_chg | 의미 |
|-------|----------------|------|
| 평가 | +42.57 | 증권 가격 변동 (핵심) |
| 환산 | -1.52 | 환율 변동 손익 |
| 매매 | +0.06 | 매매 실현손익 |
| 이자 | +0.00 | 이자수익 |
| 배당 | 0 | 배당수익 |
| 기타 | -0.01 | 기타 비용 |
| **합계** | **+41.10** | = 기준가 변동 |

**핵심**: `pl_gb='환산'`으로 FX 손익이 이미 분리되어 있어, R처럼 별도 환율 수식 없이도 분리 가능.

### 2.3 R vs Python 구현 차이

| 항목 | R (정밀) | Python (현재) |
|------|----------|---------------|
| 시간 단위 | 일별 x 종목별 | 기간 전체 합산 |
| 비중 기준 | T-1 시가평가액 / (T-1순자산 + 순설정금액) | 기간말 val.last() |
| 수익률 | 일별 수정기준가 변동률 | modify_unav_chg / val (기간합산) |
| FX 분리 | 환율수익률로 증권/환산 분리 | 미구현 |
| 누적기여도 | 경로의존적 누적 공식 | 단순 합산 |
| 보정인자 | 일별<->누적 수익률 보정 적용 | 미적용 |
| 유동성/기타 | 전체수익률 - 종목기여도합 = 잔차 | 미적용 |

## 4. R Shiny PA 핵심 로직 정리 (구현 참조용)

### 4.1 종목별 비중 (T-1 기준)

```r
weight_PA = 조정_평가시가평가액 / (순자산총액(T-1) + 순설정금액)
# SHORT: 음의 시가평가액
# 순설정금액: DWPM12880에서 해당일 설정/환매 합산
```

### 4.2 종목별 기여수익률

```r
기여수익률 = 수익률(FX_제외) * abs(weight_PA(T-1))
유동성및기타 = Return - sum(기여수익률)  # 잔차
```

### 4.3 FX 분리 (pl_gb 활용)

```r
환산 = sum(amt[pl_gb=="환산"])
총손익금액_당일_FX_adjust = 총손익금액_당일 - 환산_adjust
# 환산_adjust: USD 종목은 시가(T-1)*환율수익률 + 상호작용항
```

### 4.4 Brinson 3-Factor

```r
Allocation  = (AP비중 - BM비중) * BM수익률
Selection   = BM비중 * (AP수익률 - BM수익률)
Cross       = (AP비중 - BM비중) * (AP수익률 - BM수익률)
보정인자    = 초과수익률 / (Allocation + Selection + Cross)  # 누적보정
```

### 4.5 누적기여도 (경로의존적)

```r
sec_id기여도 = (손익금액 / 당일포트수익률) * 당일기준가증감
총손익기여도 = 누적수익률 * cumsum(sec_id기여도) / 누적기준가증감
```

## 5. 내일 구현 계획

### Phase 4: PA 정밀화 (R 동일 로직)

구현 순서:

1. **load_pa_source() 확장** — position_gb, pl_gb, crrncy_cd, os_gb 추가 SELECT
2. **일별 T-1 비중 계산** — val(T-1) / NAST_AMT(T-1), SHORT 음수 처리

4. **일별 종목 기여수익률** — 수익률 x 비중(T-1), 유동성잔차 계산
5. **누적기여도** — 경로의존적 공식 적용
6. **Brinson 3-Factor 일별화** — 일별 AP/BM 비중x수익률 → 누적
7. **검증** — sum(종목기여도) + FX + 유동성 = 포트수익률
