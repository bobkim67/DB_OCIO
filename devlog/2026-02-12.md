```
module_03_post_analysis(PA).R (500줄) - PA Shiny 모듈 UI/Server
```

### R -> Python 매핑 핵심
- `long_form_raw_data_input()` -> `data_loader.py::load_*` 시리즈
- `calculate_BM_results_bulk_for_users_input()` -> `portfolio_analytics.py` (미구현)
- `General_PA()` -> `brinson.py` (미구현)
- T-1 lag: 해외자산은 전일 가격 사용
- FX 보정: `(1+r) * (1 + FX_r * (1-hedge) * (region!="KR")) - 1`
- Brinson 3-Factor: Allocation + Selection + Cross + Correction Factor
